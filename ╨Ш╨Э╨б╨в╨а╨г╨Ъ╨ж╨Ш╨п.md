# 💬 MyChat — Инструкция по запуску

## ШАГ 1 — Установи Node.js

1. Зайди на сайт: **https://nodejs.org**
2. Нажми большую зелёную кнопку **"LTS"** — скачается установщик
3. Запусти его и нажимай "Next" до конца
4. После установки перезагрузи компьютер

---

## ШАГ 2 — Подготовь папку с чатом

1. Распакуй архив `mychat.zip` в любое место (например, рабочий стол)
2. Открой папку `mychat`
3. Внутри должны быть файлы:
   - `server.js`
   - `package.json`
   - папка `public/` с файлом `index.html`

---

## ШАГ 3 — Запусти сервер

1. Открой папку `mychat`
2. В адресной строке проводника напечатай **`cmd`** и нажми Enter
   _(откроется чёрное окно — командная строка)_
3. Введи эту команду и нажми Enter:
   ```
   npm install
   ```
   _(подождёт немного, скачает нужные файлы)_

4. Потом введи:
   ```
   npm start
   ```

5. Увидишь сообщение:
   ```
   ✅ Чат запущен!
   🌐 Открой в браузере: http://localhost:3000
   ```

---

## ШАГ 4 — Открой чат

Зайди в браузер и введи адрес:
```
http://localhost:3000
```

Введи своё имя — ты в чате! 🎉

---

## Как другие люди могут подключиться?

### В твоей домашней сети (Wi-Fi):
1. Найди свой IP-адрес: открой `cmd` и введи `ipconfig`
2. Найди строку **IPv4-адрес** — например `192.168.1.5`
3. Дай другим этот адрес: `http://192.168.1.5:3000`
4. Они должны быть подключены к тому же Wi-Fi

### Через интернет для всех (бесплатно):
Используй сервис **Railway.app**:
1. Зарегистрируйся на **https://railway.app** (через GitHub)
2. Нажми "New Project" → "Deploy from GitHub"
3. Загрузи папку mychat
4. Railway даст тебе ссылку типа `mychat.railway.app`
5. Отправь эту ссылку любому человеку — он сможет написать тебе!

---

## Если что-то не работает

- **Ошибка "npm не найден"** → Node.js не установился, попробуй переустановить
- **Страница не открывается** → убедись что в окне cmd написано "Чат запущен"
- **Другие не могут подключиться** → проверь что они в том же Wi-Fi
